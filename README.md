# Video Downloader 

VideoDownloader is a Python Script to download any video from social media platforms: 
Facebook, Twitter, Snapchat, Instagram and YouTube 

## Installation
Install [Python](https://www.python.org/ftp/python/3.10.6/python-3.10.6-amd64.exe) 3.10.6

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install libraries.

```bash
pip install yt_dlp pathlib  
```

## Usage
- Run main.py file in cmd terminal

- Enter the video url 

- The downloaded video will be under Downloads/ Folder

```cmd
python main.py

>> Enter URL:
```
